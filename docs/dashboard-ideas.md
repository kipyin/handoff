# Dashboard ideas (future)

Ideas for useful analytics to add to the Handoff Dashboard page. Not all are implemented.

- **Throughput:** Completed-per-week (already present). Optional per-helper and per-project breakdown.
- **Cycle time:** Time from creation to completion; median/percentiles by project or helper (basic cycle time exists).
- **Load:** Open handoff count by helper; "busiest week" or trend.
- **Deadline adherence:** Overdue rate (handoff items completed after deadline vs on time).
- **Export:** Optional CSV/JSON of aggregated metrics for external reporting.
